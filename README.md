\# Procesamiento de datos – PI/PID/LQR



Este repositorio contiene scripts en Python para:



\- Leer y organizar ensayos (CSV)

\- Graficar referencia vs salida

\- Calcular métricas: RMSE, overshoot, tiempo de asentamiento, etc.

\- Comparar controladores en una misma figura

\- Exportar figuras para el informe final



\## Estructura

\- `data/`: CSV por experimento

\- `scripts/`: scripts de análisis

\- `figures/`: imágenes exportadas

\- `docs/`: notas, capturas, explicación de métricas
